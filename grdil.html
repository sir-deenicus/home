Calculate Schwarzchild Blackhole Horizon, Time Dilation and Acceleration required to Hover

First, a note. I actually first wrote this code in F# and decided it was small enough to quickly port to javascript as an interactive demo. F# was extremely helpful, not just because of static typing but also because of the ability to use measurement units as types. I strongly suggest you check out the code below to get a hint of the awesomeness, you can hover over the variable names to see what the units are. I found that was very useful in helping me figure out how to get to geometric units and back in the acceleration portion. A kind of automatic dimensional analysis. Most of my time on this project was in working around javascript's poor numeric capabilities and a bit on minor styling. I also realized I've been spoiled by static typing.

F# Code

         1: [<Measure>] type m
         2: [<Measure>] type kg
         3: [<Measure>] type s
         4:  
         5: let G = 6.67e-11<m^3 *kg^-1 * s^-2>
         6: let c_light = 299792458.<m/s>
         7: let c_sq = c_light * c_light
         8: let schwarzrad = 2. * G / c_sq
         9: let toMeters (x:float) = x * 1.<m>
        10: 
        11: let sTimedil margin (mass:float<kg>) properTime =  
        12:     let sradius = mass * schwarzrad
        13:     let dist = sradius + margin
        14:     let scale =  (1. - sradius/dist) 
        15:     let radjust =  (float(dist - sradius)) |> toMeters
        16:     let mgeom = G * mass / c_sq
        17:     
        18:     //As a demonstration the unit for aGeom is [/m]. 
        19:     let aGeom = mgeom / (1.<m^2> * (float dist ** 2.) * sqrt(1. - 2. * mgeom / dist))
        20:     //But If we multiply it by c^2, we get the units for acceleration, [m/s^2]
        21:     let a_ = aGeom * c_sq
        22: 
        23:     let a = (G * mass)/(toMeters(float dist ) * radjust) 
        24:     properTime / scale,scale,a ,sradius, dist, 1.<s>*a/c_light, 3. * sradius
